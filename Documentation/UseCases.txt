PLAY AUDIO CLIP
------------------------------------------------------------------------------
	1. User navigates to categories pivot. (Main Page)
	2. User selects category. (Main Page)
	3. If catagory contains subgroups,
		Then user selects group from Catagory Page. (Catagory Page)
	4. User is forwarded to Group Page. (Catagory Page or Main Page)
	5. User navigates pivots for desired sound clip. (Group Page)
	6. User selects sound clip.(Group Page)
	6. Sound clip plays. (Group Page)
	7. Sound clip completes.(Group Page)
	
	---------------------------------------------------------------------------
		NOUN EXTRACTION 
	---------------------------------------------------------------------------
		User
		pivot
		Category
		Sound Group
		Sub Sound Group
		Sound Clip
	
	---------------------------------------------------------------------------
		SCENARIOS 
	---------------------------------------------------------------------------
		// Example with subgroup
		1. User navigates to categories pivot.
		2. User selects Actors.
		3. User navigates to Clint Eastwood pivot.
		4. User selects "A Fistful of Dollars" group.
		5. User selects " I don't think its nice, you laughin' " clip.
		6. Sound clip plays.
		7. Sound clip completes.
		
		// Example without subgroup
		1. User navigates to categories pivot.
		2. User selects Animals.
		3. User navigates to Cats pivot.
		4. User selects random meowing clip.
		5. Sound clip plays.
		6. Sound clip completes.
